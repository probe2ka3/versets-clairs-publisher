3. Purpose of the Application

The Application provides the Operator with a lightweight, browser-based interface to:

  1. Authenticate with TikTok using TikTok Login Kit (OAuth 2.0);
  2. Browse a manifest of prepared MP4 video files (hosted in the Operator’s Cloudflare R2 bucket) together with their pre-written captions;
  3. Review and, if needed, lightly edit each caption before posting;
  4. Upload the selected MP4 file to TikTok’s Inbox as a draft (using the video.upload scope), or publish it directly to the Operator’s TikTok profile (using the video.publish scope) via TikTok’s Content Posting API;
  5. Optionally schedule a future publication, processed by a Cloudflare Worker on a five-minute cron trigger.

The Application is a private creator tool. It is not a social network, marketplace, advertising network, or content recommendation system.
